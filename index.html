About This Site

This platform collects local news and presents it in a clean, screen-reader-friendly format. News data is stored in JSON files for simplicity and efficiency.

The frontend is built with React and designed for compatibility across Windows, macOS, iPhone, and Android devices.

Accessibility Features

  • Semantic HTML structure
  • Keyboard navigation support
  • Clear heading hierarchy
  • ARIA landmarks and labels
  • High-contrast readable layout

Designed to reduce information barriers and promote equal access to local news.
