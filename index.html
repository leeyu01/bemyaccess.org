About This Site

This platform collects local news and presents it in a clean, screen-reader-friendly format. No database is required; news data is stored in JSON files for simplicity and efficiency.

The frontend is built with React and designed to ensure compatibility across Windows, macOS, iPhone, and Android devices.

Accessibility Features

  • WCAG-compliant semantic HTML structure
  • Full compatibility with screen readers (NVDA, JAWS, VoiceOver, TalkBack)
  • Keyboard navigation support
  • Clear heading hierarchy
  • ARIA landmarks and labels
  • High-contrast readable layout
