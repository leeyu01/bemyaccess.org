# bemyaccess.org

Project Overview

1. Purpose

  Provide an accessible, screen-reader optimized local news and disability resource platform for individuals with   visual disabilities in Suffolk County.

2. Core Principles

  No login required
  
  No paid voice services
  
  WCAG 2.2 AA compliant
  
  Low-cost infrastructure (AWS Free Tier)
  
  JSON-based storage (no database)
  
  Daily RSS aggregation
  
  Retain only latest 90 days of content
  
  WCAG (Web Content Accessibility Guidelines) is an international standard for making websites and web       applications accessible to people with disabilities. Most public and government websites aim for WCAG 2.2 Level AA compliance.
